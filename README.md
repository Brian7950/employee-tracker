## Description: <a name="description"></a>
### Employee-Tracker allows one to use terminal to access a database through a mysql connection and use inquirer to select options and manipulate data as needed. 

## Installation: <a name="installation"></a>
### For installation, user will need to install dependencies via npm i within node.js on CLI. User will also need to configure mysql connection to allow for database communication. 

## Technologies <a name="technologies"></a>
### JavaScript, Node.js, MySql, dotenv, inquirer, mysql2

## Usage: <a name="usage"></a>
### Employee-Tracker is simple application that can be used from ones command line to access an employee database and view employees, roles, and departments. Users can also add additional roles, employees, and departments as needed as well as edit them.
